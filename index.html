Projects

Karaton

The Karaton project is an educational game aimed at helping young children aged 6 - 10 years old practicing their dyslexia exercises to help improve their word comprehension abilities. The game was 1 part of a larger project, with the game aimed at providing motivation for the children to keep doing their exercises to gather resources for crafting items, whilst the results were sent to an external website that parents and speech therapists could view their performance in real time, customizing the words they practiced on within the game.
I joined onto the Karaton project initially as an intern, however the project ended up having to be scrapped and restarted. With this in mind, I helped rescope the project into something that was more achievable given the time frame and platform. I ended up leading the project and creating the majority of it from scratch again. Some of the key features I worked on within the game are:
  • Support for cloud saving across multiple platforms using Gamesparks.
  • Delivered initial version of the game built from scratch within 6 months.
  • Implementing core controller for player and interaction system with objects in the world.
  • Implemented build mode, where players could place object into the world and save it into the cloud to view on another platform.
  • Created base framework for all other exercise minigames for external developers to use when creating minigames.
  • Implemented and iterated on scrum based workflow within the company.

Vicious Voxels

A small First Person Shooter game I created for a school project. Implemented both A* pathfinding and Genetic Algorithms into the game to help enhance the gameplay.
A wave based shooter that had 3 different enemy types, the game operated on a simple rock, paper, scissors mechanic, with the 3 guns available to the player acting as a counter to the enemy types. The Genetic Algorithm would evolve the make up of the next waves based on how the player fought the last one. The following metric were measured:
  • Successful shots fired by the player (meaning that they exploited a weakness)
  • Time taken to clear the wave
  • Damage taken by player
  • Average lifetime of each enemy type
A link to the final paper explaining it in depth can be found here.
